# Канонические принципы — company_to_worker

## Основные коды

- `CtoW01` — Прозрачная процедура собеседований при приеме на работу с обратной связью
- `CtoW02` — Справедливая оплата труда, отражающая вклад, квафилификацию и риски
- `CtoW03` — Безопасные и по возможности комфортные условия труда
- `CtoW04` — Забота о здоровом образе жизни и личном благополучии работников
- `CtoW05` — Прозрачная процедура при увольнении с обратной связью
- `CtoW06` — Отсутствие дискриминации по этническим, гендертным или возрастным признакам
- `CtoW07` — Создание условий для приема на работу людей с ограниченными возможностями
- `CtoW08` — Более гибкие условия найма для одиноких родителей, студентов и пожилых людей
- `CtoW09` — Регулярный сбор обратной связи от работников по поводу возможных проблем
- `CtoW10` — Свобода объединений работников в профсоюз или другие ассоциации для защиты своих интересов
- `CtoW99` — иной принцип, не покрытый основным набором.

## Правила использования

1. Сначала соотноси фрагмент с `CtoW01-CtoW10`.
2. `CtoW99` используй только если не подходит ни один канонический код.
3. Для `CtoW99` обязательно заполняй:
   - `other_principle_label`
   - `other_principle_explanation`
