# Канонические принципы — worker_to_manager
## Основные коды

- `WtoM01` — Сотрудник должен уважительно общаться с руководителем (вежливость, корректность, профессиональный и доброжелательный тон)
- `WtoM02` — Сотрудник должен проявлять честность и открытость по отношению к руководителю (недопустимо скрывать значимую информацию и вводить руководителя в заблуждение для достижения личной выгоды, напр., отлынивания от выполения работы)
- `WtoM03` — Сотрудник должен добросовестно выполненять поручений руководителя
- `WtoM04` — В случае несогласия сотруднику следует выражать свое мнение вежливо, открыто и аргументировано. Неэтично скрывать несогласие.
- `WtoM05` — Недопустимость подхалимства и манипуляций со стороны сотрудников. Необходимо поддержание добросовестных нейтральных профессиональных отношений.
- `WtoM06` — Сотруднику следует отказываться выполнять неэтичные или незаконные поручения руководителя
- `WtoM99` — иной принцип, не покрытый основным набором.

## Правила использования

1. Сначала соотноси фрагмент с `WtoM01-WtoM06`.
2. `WtoM99` используй только если не подходит ни один канонический код.
3. Для `WtoM99` обязательно заполняй:
   - `other_principle_label`
   - `other_principle_explanation`
