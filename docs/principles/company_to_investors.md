# Канонические принципы — company_to_investors

## Основные коды

- `CtoI01` — Равные права всех акционеров и кредиторов на доступ к информации
- `CtoI02` — Организация собраний акционеров, максимально облегчающая доступ каждого акционера к участию в собрании
- `CtoI03` — Максимально полное и своевременное раскрытие всей существенной информации для акционеров и кредиторов
- `CtoI04` — Прозрачное проведение собраний и принятие решений компанией
- `CtoI05` — Предотвращение инсайдерской торговли 
- `CtoI06` — Обоснованная дивидендная политика
- `CtoI07` — Защита прав миноритарных акционеров 
- `CtoI08` — Справедливое отношение к кредиторам компании при погашении долга
- `CtoI99` — иной принцип, не покрытый основным набором.

## Правила использования

1. Сначала соотноси фрагмент с `CtoI01-CtoI08`.
2. `CtoI99` используй только если не подходит ни один канонический код.
3. Для `CtoI99` обязательно заполняй:
   - `other_principle_label`
   - `other_principle_explanation`
