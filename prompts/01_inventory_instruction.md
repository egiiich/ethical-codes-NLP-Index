# Instruction 01 — Inventory of the Corpus

## Task
Провести инвентаризацию корпуса корпоративных кодексов этики в репозитории.

## Context
- Материалы находятся в GitHub-репозитории.
- Этот этап нужен только для формирования реестра документов.
- На этом этапе не нужно делать содержательный анализ тематик.

## Required actions
1. Просмотреть материалы в репозитории.
2. Выявить документы, являющиеся:
   - code of ethics
   - code of conduct
   - business conduct guidelines
   - иными функциональными эквивалентами
3. Для каждого документа добавить запись в `registry/codes_index.csv`.

## Required fields
- `document_id`
- `company_name`
- `country`
- `industry`
- `source_path`
- `source_language`
- `file_format`
- `document_type`
- `review_status`

## Rules
- Не присваивать статусы тематической релевантности на этом этапе.
- Если метаданные неочевидны, указывать наиболее вероятное значение и отмечать неопределённость для человека.
- Для новых документов использовать `review_status = not_started`.

## Output
1. Обновлённый `registry/codes_index.csv`
2. Краткое резюме:
   - какие документы добавлены;
   - какие записи требуют ручной проверки метаданных.
